# Karin_chatbot
Simple Chatbot for nasi padang restaurant

This chatbot is created using dialogflow.

How to use this sample?
 - Login and create agent in dialogflow
 - import this file
 - deploy fulfillment
 - integration chatbot in telegram or others media
